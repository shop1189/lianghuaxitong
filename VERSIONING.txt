Git 基线与回退（longxia_system）
================================

已建立仓库与标签，便于「只回代码、不动本地数据」。

常用命令
--------
  git tag -l                          # 查看标签
  git log --oneline -5                # 最近提交

回到标记的基线（仅影响已纳入 Git 的代码文件）：
  git checkout baseline-2026-04-18

回到当前分支最新：
  git checkout main
