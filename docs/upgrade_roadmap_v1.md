## 第 0 阶段 · 样本观察期

- 虚拟单 / 实验轨积累样本；仪表盘与 `trade_memory` 可对账。  
- **不**急于上大改；热修复以「可观测、可回滚」为原则。

## 第 1 阶段 · 工程对齐（与合并稿选项 **A** 一致）

- **同源**：信号生成、SL/TP 档位、虚拟/实验平仓判定与线上一致（已起步：`utils/trade_exit_rules.py`）。  
- **最小回测入口**（后续迭代）：同一套 `first_exit_tick` / 扩展接口跑历史 K 线或回放。

## 第 2 阶段 · 回测 / 记忆

- 回测口径与手续费、bar 源**冻结基线**后再调参。  
- 「行情记忆」、马尔可夫状态等：**按需**接入，不捆绑一次性上线。

## 第 3 阶段 · 自动化与重型工具

- FinRL / LLM 反思 / 动态仓位等：见 `UPGRADE_PLAN` 阶段 C～E；**A/B/C 未稳前**不默认开 GPU/HF 长推理。

## 第 4 阶段 · 体验与书库（与合并稿选项 **B** 可合并）

- 指标展示、书库扩展、SL/TP 页面对照：**产品向**，与工程阶段可并行立项但**范围单开**。
